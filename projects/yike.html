Yike — Plataforma para agentes sanitarios

Problema

Los agentes sanitarios trabajan en zonas sin conectividad, lo que dificulta el registro y seguimiento de datos.

Solución

Diseñé una aplicación offline-first que permite registrar datos localmente y sincronizarlos cuando hay conexión.

Arquitectura

  • Frontend: React Native
  • Backend: FastAPI
  • Base local: SQLite
  • Sync engine custom

Decisiones técnicas

Se implementó un modelo de sincronización basado en timestamps y resolución de conflictos para garantizar consistencia.
